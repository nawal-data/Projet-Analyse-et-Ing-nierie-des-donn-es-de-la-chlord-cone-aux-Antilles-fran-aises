🧼 Étapes principales :
1. Nettoyage des données
•	Harmonisation des colonnes
•	Nettoyage des valeurs textuelles
•	Conversion des dates
•	Traitement des valeurs manquantes
•	Recodage des types de sols


2. Analyse exploratoire
•	Statistiques globales
•	Distribution du taux de chlordécone
•	Analyse par commune et type de sol
•	Matrice de corrélation


4. Visualisations
•	Histogrammes
•	Boxplots
•	Heatmap Folium
•	Carte X-Y colorée


5. Analyse avancée
•	ACP (réduction de dimension)
•	Clustering K-means
•	Modèle Random Forest


Résultats clés
•	Les Andosols sont les sols les plus contaminés
•	Le Nord Atlantique est la zone la plus touchée
•	La topographie influence fortement la rétention du pesticide
•	Le clustering identifie un cluster critique (zones plates + Andosols)
•	Le modèle prédictif permet d’anticiper les zones à risque
